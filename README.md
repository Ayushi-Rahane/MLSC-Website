# MLSC Website — Frontend-first Scaffold

This repository is a frontend-first scaffold for a Microsoft Learn Student Club (MLSC) website.

Features
- React (Vite)
- Tailwind CSS
- React Router (client-side pages)
- Backend scaffold with Express (for later expansion)
- Cross-platform one-command setup for contributors

Quick start (macOS / Linux)
```bash
./setup.sh
```

Quick start (Windows PowerShell)
```powershell
.\setup.ps1
```

What the scripts do
- Install frontend and backend dependencies automatically
- Create `.env` placeholders if missing
- Start the frontend dev server (and backend in parallel when available)

Project layout (top-level)
- `frontend/` — React app (Vite + Tailwind)
- `backend/` — Express scaffold

Contributing
- Open an issue or PR with proposed UI changes.
- Run `./run.sh` (or `run.ps1`) to start the project after setup.

If you need help, contact the repo maintainers.
